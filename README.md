Coronal Suture Extraction

Problem

Stereotactic neurosurgery requires precise localization to minimize damage to healthy brain tissue, but current approaches still rely heavily on surgeon experience and have limited adaptability to complex anatomical variations. This project aims to develop an AI-assisted system to improve surgical path planning by enhancing accuracy and consistency.

Method

This project utilizes brain CT images to perform skull and hematoma segmentation, followed by anatomical analysis to identify safe entry regions. Based on these results, an automated path planning framework is developed to generate optimal surgical trajectories while avoiding high-risk structures.

Results

The system successfully generates consistent and anatomically aligned surgical paths, reducing potential risks and providing a more standardized approach to path planning.
